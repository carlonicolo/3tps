# Linux course

## Commands

```bash
$ pwd
```

```bash
$ ls
$ ls -a
$ ls -l
$ ll
```

```bash
$ mkdir folderName
```

```bash
$ touch filename.txt
```

```bash
$ cat filename.txt
```

```bash
$ cp /home/daniel/filename.txt .
```  

Copy a file inside a directory - cp [sourceFolder] [destinationFolder]  
```bash
$ cp /home/daniel/filename.txt /home/daniel
```
